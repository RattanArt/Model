# Concept : ตรรกศาสตร์ (Logic)

## ประเด็น
- ความหมายของตรรกศาสตร์
- ประพจน์
- ตัวเชื่อมตรรกะ
- การวิเคราะห์ค่าความจริง
- ตารางค่าความจริง
- การอ้างเหตุผล
- การพิสูจน์

## Truth Tables

```mermaid
mindmap
	root((Truth Tables))
		Negation  ¬ 
			Tables of ¬p
			Oppsite				
				Double Negation
		Conjunction  ∧
			Tables of p ∧ q
			Only true when both are true
		Disjunction  ∨
			Tables of p ∨ q
			True if at least one is true
		Implication  ⇒
			Tables of p ⇒ q
			False only when p is true and q is false
		Biconditional  ⇔ 
			Tables of p ⇔ q
			True when p and q have same truth value
		Combined Connectives
			Example 
				id1["(p ∧ q)  ⇒ r"]
				id1[" ¬p ⇔ (q ∨ r)"]
				id1["[(q ⇔ s) ∧ r] ∨ [(q ⇒ s)∨ r]"]
			Can be expanded into full truth table
```
